Install Chef DK

First thing first: install Chef DK. It is a package which provides developers with all the tools to start developing and testing cookbooks.
Berkshelf
Test Kitchen
Foodcritic
and many others
- Berkshelf, dependency manager, i.e. "for cookbook A to work, cookbook B has to be installed"
- Test Kitchen, integration testing framework
- Foodcritic, linting tool for doing static code analysis on cookbooks
